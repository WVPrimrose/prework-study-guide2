          Examples

        • colors

          Different Ways

        1. Inline
          2. Styling CCS rules directly into the HTML file's elements. Problem is you have to style each and every individual HTML tag.
        2. Internal
          3. Styling CCS rules directly into the HTML file's head. If you have multiple styling elements, this makes changes harder to navigate
        3. External
          4. Stying CCS rules in a separate file that'll be linked to the HTML file's head. This is the prefered method because this makes changes
          much easier in an organizaed fashion.
